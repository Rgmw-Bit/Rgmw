# Rgmw

Rgmw es una inteligencia artificial con personalidad propia, diseñada para interactuar, asistir y evolucionar con el tiempo.

## Descripción

Rgmw es un asistente digital con identidad única, inspirado en la personalidad de un gato profesional. Este proyecto tiene como objetivo crear una inteligencia artificial que pueda ayudar, interactuar y mejorar progresivamente.

## Estado del proyecto

En desarrollo.

## Objetivo

Convertir a Rgmw en un asistente completo accesible desde la web y futuras aplicaciones.
